## [1.2.0] - 2026-03-10

### 신규 기능
- **대출 상환 스케줄** — 대출별 회차별 상환 내역을 한눈에 확인할 수 있습니다. 납부 완료 / 예정 / 연체 상태가 자동으로 표시됩니다
- **정기 이체** — 매월 또는 매년 반복되는 이체(계좌→계좌, 계좌→카드)를 등록하면 앱 실행 시 자동으로 처리됩니다
- **대출/고정비용 삭제 시 연쇄 처리** — 삭제 시 관련 기록도 함께 삭제할 수 있으며, 자동 차감된 금액의 복원 여부를 선택할 수 있습니다
- **밀린 회차 자동 처리** — 앱을 오래 열지 않았어도, 재실행 시 밀린 대출/고정비용/정기이체 회차가 자동으로 소급 처리됩니다
- **백업 복원 진행률 표시** — 백업 복원 중 진행 상태를 확인할 수 있습니다

### 보안 강화
- 비밀번호 검증 해시의 반복 횟수를 10,000회에서 100,000회로 강화했습니다. 기존 사용자는 다음 로그인 시 자동으로 업그레이드됩니다
- 백업 파일 생성 시 원자적 쓰기를 적용하여 백업 중 앱이 종료되어도 파일이 손상되지 않습니다
- 백업 복원 시 파일 헤더 검증을 강화하여 손상된 백업 파일에 대해 명확한 오류 메시지를 표시합니다

### 버그 수정
- 자동 차감 중 오류 발생 시 이중 차감이 발생할 수 있는 문제 수정
- 일부 데이터 파일 손상 시 앱 전체가 로드되지 않던 문제 수정 (부분 복구 가능)
- 고정비용/정기이체 실행 중 앱이 종료되면 재시작 시 중복 기록이 생성되던 문제 수정
- 동일 결제일의 대출이 여러 건일 때 기록이 잘못 매칭되던 문제 수정
- 대출 조건 변경 후 상환 기록의 납부 상태가 유실되던 문제 수정
- 정기 이체 화면에서 다크모드 시 버튼 텍스트가 보이지 않던 문제 수정
- 대출 화면에서 하드코딩된 한국어가 번역 키로 교체되었습니다
- 비밀번호 변경 중 오류 발생 시 데이터 복원 안전성 개선
- 앱 재암호화(비밀번호 변경) 중 크래시 발생 시 자동 복구 기능 추가

### UI/UX
- 대출 상환 방식 변경은 삭제 후 재등록으로 안내 (데이터 무결성 보호)
- 다크모드에서 일부 화면의 하드코딩된 색상을 테마 토큰으로 교체

## [1.1.0] - 2026-02-27

### 신규 기능
- **카드 정보 편집** — 등록된 카드의 이름, 결제일, 연결 계좌, 색상을 수정할 수 있습니다
- **체크카드 자동 차감** — 체크카드로 결제하면 연결된 계좌에서 즉시 잔액이 차감됩니다. 기록 수정/삭제 시 잔액도 자동으로 복원됩니다
- **홈 이체 바로가기** — 홈 화면의 빠른 입력에 이체 버튼이 추가되었습니다
- **고정비용 시스템** — 매월 또는 매년 반복되는 고정 지출을 등록하면 앱 실행 시 자동으로 기록됩니다

### 버그 수정
- 웹 결제 후 구독이 활성화되지 않는 문제 수정
- 백업 복원 후 데이터가 표시되지 않는 문제 수정
- 백업 복원 후 자산이 이중으로 차감되는 문제 수정
- 일반 계좌에서 마이너스 잔액이 허용되던 문제 수정
- 마이너스 통장에서 한도를 초과할 수 있던 문제 수정

### UI/UX
- iOS 모달 상단의 불필요한 여백 제거 (19개 모달)

## [1.0.0] - 2026-02-25

SYBA의 첫 번째 정식 출시 버전입니다.

### 가계부
- 수입/지출 기록 — 카드, 현금, 라이트닝, 온체인 결제 수단 지원
- 모든 금액을 비트코인(sats) 단위로 자동 환산 — 기록 날짜의 종가 기준
- 카테고리별 분류 (기본 카테고리 + 사용자 정의 카테고리 그룹 생성 가능)
- CSV 일괄 가져오기 — SYBA 전용 양식으로 여러 건의 기록을 한 번에 가져오기

### 자산 관리
- 은행 계좌 등록 (일반 계좌, 마이너스 통장)
- 비트코인 지갑 등록 (온체인, 라이트닝)
- 계좌 간 이체 (원자적 처리, 실패 시 자동 복원)
- 선불카드 충전 (계좌에서 선불카드로 잔액 이동)
- 실시간 비트코인 시세 — 지역별 API 자동 선택 (한국: Upbit, 미국: Coinbase, 아르헨티나/일본: OKX)

### 부채 관리
- 대출 관리 — 원금균등, 원리금균등, 만기일시 상환 방식 지원
- 할부 관리 — sats 환산 포함
- 카드 결제 예정액 자동 계산
- 월별 자동 차감 스케줄링 — 앱 실행 시 해당 월의 상환금 자동 기록

### 보안
- AES-256-CBC로 모든 금융 데이터 암호화
- PBKDF2 키 도출 (100,000회 반복)
- iOS Keychain / Android Keystore에 암호화 키 보관
- 앱 재설치 감지 시 자동 초기화 (생체인증 우회 방지)
- 암호화된 백업 파일 생성 및 복원

### 차트/통계
- 카테고리별 지출 비율 (파이차트)
- 수입 vs 지출 흐름 (막대/꺾은선 그래프)
- 월별 지출 추이
- 순자산 변화 추이

### 국제화
- 4개 언어 지원 — 한국어, English, Español (AR), 日本語
- 지역별 은행/카드사 목록 자동 적용
- 다통화 대응 (KRW 하드코딩 제거)

### 인증
- 비밀번호 기반 로그인 (12자 이상, 대소문자/숫자 포함)
- 생체인증 (Face ID / 지문) 지원
- 5회 연속 실패 시 5분간 잠금
- 자동 잠금 타이머 (즉시, 1분, 5분, 15분, 30분)

### 기타
- 다크모드 / 라이트모드 / 시스템 자동
- 프리미엄 구독 — Lightning Network를 통한 결제, LNURL-auth 기반 익명 인증
- 표시 단위 전환 (원화 ↔ sats)
- 개인정보 처리방침 공개
