## [1.2.0] - 2026-03-11

### 신규 기능
- **반복 이체** — 계좌 간 또는 계좌에서 카드로의 자동 이체를 설정할 수 있습니다. 시작일, 주기, 종료 조건을 지정하면 앱 실행 시 자동으로 처리됩니다
- **대출 상환 일정 자동 생성** — 대출 등록 시 전체 상환 일정이 미리 생성됩니다. 각 건의 상태(납입됨/예정/연체)를 색상으로 구분해 대출 상세 화면에서 확인할 수 있습니다
- **카드사별 할부 금리 자동 적용** — 할부 등록 시 카드사 기본 금리가 자동으로 입력됩니다. 직접 수정도 가능합니다
- **백업 복원 진행률 표시** — 백업 복원 중 진행 상황을 프로그레스 바로 확인할 수 있습니다

### 보안
- **암호화 키 도출 방식 강화** — PBKDF2의 해시 알고리즘을 SHA-1에서 SHA-256으로 업그레이드했습니다. 기존 사용자는 로그인 시 자동으로 마이그레이션됩니다
- **비밀번호 변경 안전장치** — 비밀번호 변경 중 재암호화가 실패하면 이전 비밀번호로 자동 복원됩니다. 처리 중 "앱을 닫지 마세요" 경고가 표시됩니다
- **파일 저장 원자성 보장** — 데이터 저장 시 임시 파일에 먼저 기록한 뒤 원본을 교체합니다. 앱이 갑자기 종료되어도 데이터가 손상되지 않습니다
- **재암호화 크래시 복구** — 재암호화 도중 앱이 종료되더라도 다음 실행 시 자동으로 완료됩니다

### 버그 수정
- 월별 자동 차감이 2월 등 짧은 달을 건너뛰던 문제 수정
- 대출 상환 기록이 중복 생성되던 문제 수정
- 앱 크래시 시 대출 거래가 유실되던 문제 수정
- 활성 할부가 있는 카드를 삭제할 수 있던 문제 수정
- 대출 삭제 시 연결된 자산 잔액이 복원되지 않던 문제 수정
- 오프라인 상태에서 가격 동기화가 실패하던 문제 수정
- 만료된 반복 거래가 계속 실행되던 문제 수정
- 다크모드에서 일부 색상이 하드코딩되어 보이지 않던 문제 수정
- 대출 화면의 한국어 하드코딩 라벨을 다국어 키로 교체

### UI/UX
- **삭제 확인 대화상자 개선** — 대출, 반복 거래, 반복 이체 삭제 시 "관련 기록도 함께 삭제할지" 3단계 선택지가 표시됩니다 (취소 / 기록 삭제 / 기록 유지)

## [1.0.0] - 2026-02-25

SYBA의 첫 번째 정식 출시 버전입니다.

### 가계부
- 수입/지출 기록 — 카드, 현금, 라이트닝, 온체인 결제 수단 지원
- 모든 금액을 비트코인(sats) 단위로 자동 환산 — 기록 날짜의 종가 기준
- 카테고리별 분류 (기본 카테고리 + 사용자 정의 카테고리 그룹 생성 가능)
- CSV 일괄 가져오기 — SYBA 전용 양식으로 여러 건의 기록을 한 번에 가져오기

### 자산 관리
- 은행 계좌 등록 (일반 계좌, 마이너스 통장)
- 비트코인 지갑 등록 (온체인, 라이트닝)
- 계좌 간 이체 (원자적 처리, 실패 시 자동 복원)
- 선불카드 충전 (계좌에서 선불카드로 잔액 이동)
- 실시간 비트코인 시세 — 지역별 API 자동 선택 (한국: Upbit, 미국: Coinbase, 아르헨티나/일본: OKX)

### 부채 관리
- 대출 관리 — 원금균등, 원리금균등, 만기일시 상환 방식 지원
- 할부 관리 — sats 환산 포함
- 카드 결제 예정액 자동 계산
- 월별 자동 차감 스케줄링 — 앱 실행 시 해당 월의 상환금 자동 기록

### 보안
- AES-256-CBC로 모든 금융 데이터 암호화
- PBKDF2 키 도출 (100,000회 반복)
- iOS Keychain / Android Keystore에 암호화 키 보관
- 앱 재설치 감지 시 자동 초기화 (생체인증 우회 방지)
- 암호화된 백업 파일 생성 및 복원

### 차트/통계
- 카테고리별 지출 비율 (파이차트)
- 수입 vs 지출 흐름 (막대/꺾은선 그래프)
- 월별 지출 추이
- 순자산 변화 추이

### 국제화
- 4개 언어 지원 — 한국어, English, Español (AR), 日本語
- 지역별 은행/카드사 목록 자동 적용
- 다통화 대응 (KRW 하드코딩 제거)

### 인증
- 비밀번호 기반 로그인 (12자 이상, 대소문자/숫자 포함)
- 생체인증 (Face ID / 지문) 지원
- 5회 연속 실패 시 5분간 잠금
- 자동 잠금 타이머 (즉시, 1분, 5분, 15분, 30분)

### 기타
- 다크모드 / 라이트모드 / 시스템 자동
- 프리미엄 구독 — Lightning Network를 통한 결제, LNURL-auth 기반 익명 인증
- 표시 단위 전환 (원화 ↔ sats)
- 개인정보 처리방침 공개
