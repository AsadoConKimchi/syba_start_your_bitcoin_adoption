# SYBA 사용 가이드

SYBA의 모든 기능을 단계별로 설명합니다.

---

## 1. 시작하기

### 첫 실행

앱을 처음 실행하면 두 가지 선택지가 표시됩니다:

- **새로 시작** — 비밀번호를 설정하고 새로운 가계부를 시작합니다
- **백업 파일로 복구** — 기존에 만든 `.enc` 백업 파일을 선택하여 데이터를 복원합니다

### 비밀번호 설정

"새로 시작"을 선택하면 비밀번호를 설정합니다.

비밀번호 요구사항:
- 대문자 1개 이상
- 소문자 1개 이상
- 숫자 1개 이상
- 최소 12자 이상

> **주의**: 비밀번호를 잊어버리면 복구할 수 없습니다. 백업 파일이 있어야 데이터를 되찾을 수 있으므로, 비밀번호를 안전한 곳에 보관하세요.

### 생체인증 등록

비밀번호 설정 후, 기기가 Face ID나 지문 인식을 지원하면 생체인증 등록 화면이 나타납니다.

- **활성화** — 다음 로그인부터 비밀번호 대신 생체인증으로 빠르게 잠금 해제
- **나중에** — 건너뛰기 (설정에서 언제든 활성화 가능)

---

## 2. 홈 화면

로그인하면 홈 화면이 표시됩니다. 위에서 아래로:

- **오늘의 요약** — 오늘의 수입과 지출 합계
- **이번 달 현황** — 월 수입, 지출, 순저축 표시 (흑자면 초록, 적자면 빨강)
- **순자산 차트** — 순자산 변화 추이 (프리미엄)
- **카드 결제 예정** — 등록된 카드별 다음 결제일, 결제 예정 금액, 일시불/할부 구분
- **빠른 입력** — 지출 추가, 수입 추가, 이체 버튼
- **오늘의 기록** — 최근 기록 5건 미리보기
- **비트코인 시세** — 현재 BTC/KRW 시세와 김치 프리미엄

화면을 아래로 당기면 새로고침됩니다.

---

## 3. 계좌/카드 등록

기록을 시작하기 전에 계좌와 카드를 등록하면, 기록 시 자산이 자동으로 연동됩니다.

### 은행 계좌 등록

자산 탭 → 우측 상단 **+** 버튼:

- **계좌 종류**: 일반 계좌 또는 마이너스 통장
- **은행**: 지역에 맞는 은행 목록에서 선택
- **계좌 이름**: 자유롭게 입력 (예: "월급 통장", "비상금")
- **현재 잔액**: 현재 잔액 입력
- **한도** (마이너스 통장만): 마이너스 한도 입력

### 비트코인 지갑 등록

자산 탭 → **+** 버튼 → 자산 종류를 "비트코인"으로 선택:

- **지갑 종류**: 온체인 또는 라이트닝
- **지갑 이름**: 자유롭게 입력 (예: "콜드 월렛", "일상 결제")
- **현재 잔액**: sats 단위로 입력

### 카드 등록

설정 → 카드 관리 → **카드 추가** 또는 지출 기록 시 "카드 추가":

- **카드사**: 지역에 맞는 카드사 목록에서 선택
- **카드 종류**: 신용카드 또는 체크카드
- **카드 이름**: 자유롭게 입력
- **결제일**: 카드 대금 결제일
- **결제 계좌** (선택): 연결 계좌 선택 시 체크카드는 결제 즉시 잔액 차감
- **색상**: 구분을 위한 색상 선택

등록 후 카드 정보는 설정 → 카드 관리에서 편집할 수 있습니다 (카드사/종류는 변경 불가).

---

## 4. 수입/지출 기록

### 지출 기록하기

홈 화면의 **지출 추가** 또는 기록 탭의 빨간 **-** 버튼:

1. **금액** 입력
2. **카테고리** 선택 (식비, 교통, 쇼핑 등)
3. **결제 수단** 선택:
   - 카드 (등록된 카드 목록)
   - 현금
   - 라이트닝
   - 온체인
4. **날짜** 선택 (기본: 오늘)
5. **메모** (선택)
6. **저장**

저장하면:
- 기록 목록에 추가
- 카드 결제 시 → 해당 카드의 결제 예정액에 반영
- 체크카드 결제 시 → 연결 계좌 잔액 자동 차감
- 현금 결제 시 → 선택한 계좌 잔액 차감
- 해당 날짜의 비트코인 시세로 sats 환산 금액 자동 계산

### 수입 기록하기

홈 화면의 **수입 추가** 또는 기록 탭의 초록 **+** 버튼:

1. **금액** 입력
2. **카테고리** 선택 (급여, 용돈, 투자 등)
3. **입금 계좌** 선택
4. **날짜** 선택
5. **메모** (선택)
6. **저장** → 선택한 계좌 잔액에 자동 반영

### 기록 수정/삭제

기록 탭에서 기록을 탭하면 수정 화면으로 이동합니다:
- 금액, 카테고리, 결제 수단, 날짜, 메모 수정 가능
- 삭제 시 연결된 자산 변동도 함께 되돌려집니다

### CSV 일괄 가져오기

여러 건의 기록을 한 번에 가져올 수 있습니다:

1. 설정 → **CSV 템플릿 다운로드** — 등록된 계좌/카드/대출/할부 이름이 안내된 SYBA 전용 양식을 받습니다
2. 엑셀이나 스프레드시트에서 양식에 맞게 기록 작성
3. 설정 → **CSV 가져오기** → 파일 선택 → 미리보기 확인 → 가져오기
4. 가져온 기록의 계좌 잔액이 자동으로 반영됩니다

## 5. 부채 관리

### 대출 등록

부채 탭 → 우측 상단 **+** → **대출 추가**:

- **대출명**: 자유롭게 입력 (예: "주택담보대출", "학자금")
- **금융기관**: 입력
- **상환 방식**: 원금균등, 원리금균등, 만기일시 중 선택
- **대출 원금**: 총 대출 금액
- **연 이자율**: % 단위
- **대출 기간**: 개월 수
- **시작일**: 대출 시작 날짜
- **결제 계좌** (선택): 연결 시 매월 자동 차감

등록 후:
- 매월 앱 실행 시 해당 월의 상환금이 자동으로 기록됩니다
- 연결 계좌가 있으면 잔액에서 자동 차감됩니다
- 잔액이 부족하면 알림이 표시되고 자동 차감이 보류됩니다

### 할부 등록

부채 탭 → **+** → **할부 추가**:

- **상점명**: 구매처
- **카드**: 결제 카드 선택
- **총 금액**: 할부 총액
- **할부 개월 수**: 개월 수
- **무이자 여부**: 무이자 또는 이자율 입력
- **시작일**: 할부 시작 월

등록 후 매월 할부금이 카드 결제 예정액에 자동 반영됩니다.

### 대출/할부 상세

부채 탭에서 대출이나 할부를 탭하면 상세 화면으로 이동합니다:
- 전체 상환 스케줄 확인
- 남은 원금, 납부 현황, 진행률 확인
- 수정 및 삭제 가능

## 6. 자산 관리

### 자산 현황 확인

자산 탭에서 전체 자산을 한눈에 확인:

- **총 자산**: 법정화폐 + 비트코인 합계
- **비트코인 비율**: 전체 자산 중 비트코인 비중을 프로그레스 바로 표시
- **법정화폐 자산**: 계좌별 잔액
- **비트코인 자산**: 지갑별 잔액 (sats + 원화 환산)

각 자산을 탭하면 상세 화면에서 수정/삭제할 수 있습니다.

### 계좌 간 이체

홈 화면의 **이체** 버튼 또는 자산 탭의 이체 아이콘:

1. **출금 계좌** 선택
2. **입금 계좌** 선택
3. **금액** 입력
4. **이체** → 두 계좌의 잔액이 동시에 반영됩니다

이체가 실패하면 자동으로 되돌려집니다 (원자적 처리).

---

## 7. 고정비용 (반복 지출)

매월 또는 매년 반복되는 고정 지출을 등록하면 앱 실행 시 자동으로 기록됩니다.

### 고정비용 등록

설정 → **고정비용 관리** → **추가**:

- **이름**: 지출 항목명 (예: "넷플릭스", "자동차 보험")
- **금액**: 매 주기 금액
- **카테고리**: 분류
- **결제 수단**: 카드 또는 계좌
- **주기**: 매월 또는 매년
- **실행일**: 매월 몇 일에 자동 기록할지

등록된 고정비용은 해당 날짜에 앱을 실행하면 자동으로 지출 기록이 생성되고, 연결된 계좌/카드에 반영됩니다.

---

## 8. 차트/통계

기록 탭에서 월별 차트를 확인할 수 있습니다:

- **카테고리별 파이차트** — 이번 달 지출이 어디에 집중되었는지 비율로 표시
- **지출 추이** — 월별 지출 흐름을 꺾은선 그래프로 표시

홈 화면에서:
- **순자산 차트** — 순자산(자산 - 부채) 변화 추이

과거 월 데이터 조회와 차트는 프리미엄 구독이 필요합니다.

---

## 9. 백업/복원

### 백업 만들기

설정 → **백업** (프리미엄 기능):

- **기기에 저장** — 기기 내 원하는 위치에 `.enc` 파일 저장
- **외부로 공유** — 메신저, 클라우드 등으로 파일 전송

백업 파일은 AES-256으로 암호화됩니다. 비밀번호 없이는 열 수 없습니다.

### 백업에서 복원

앱 첫 실행 화면 → **백업 파일로 복구**:

1. `.enc` 파일 선택
2. 백업 당시의 비밀번호 입력
3. 복원 완료 → 모든 기록, 계좌, 카드, 대출, 할부가 복원됩니다

> 정기적으로 백업하는 것을 권장합니다. 기기 분실이나 앱 삭제 시 백업 파일이 유일한 복구 수단입니다.

## 10. 설정

### 표시 단위

원화(KRW)와 사토시(sats) 중 기본 표시 단위를 선택할 수 있습니다. 전환하면 앱 전체에 즉시 반영됩니다.

### 테마

라이트, 다크, 시스템(자동) 중 선택할 수 있습니다.

### 언어/지역

- **언어**: 한국어, English, Español (AR), 日本語
- **지역**: 한국, 미국, 아르헨티나, 일본 — 지역에 따라 은행/카드사 목록과 시세 API가 자동 변경됩니다

### 보안 설정

- **비밀번호 변경**: 현재 비밀번호 확인 후 새 비밀번호 설정 (모든 데이터가 새 키로 재암호화됩니다)
- **생체인증**: Face ID / 지문 인식 활성화/비활성화
- **자동 잠금**: 앱을 떠난 후 자동 잠금까지의 시간 설정 (즉시, 1분, 5분, 15분, 30분, 사용 안 함)
- **지금 잠금**: 즉시 앱을 잠그고 로그인 화면으로 이동

### 카테고리 관리

지출/수입 카테고리를 직접 만들고 관리할 수 있습니다:
- 카테고리 그룹 생성
- 각 그룹에 이름, 이모지, 색상 지정
- 활성 그룹 선택 시 기록 화면에 즉시 반영

### 알림

- **일일 알림**: 매일 지정 시간에 기록 알림
- **구독 만료 알림**: 프리미엄 구독 만료 전 알림

---

## 11. 프리미엄 구독

일부 기능은 프리미엄 구독이 필요합니다:
- 자산 탭
- 부채 탭
- 과거 월 기록 조회 / 차트
- 순자산 차트
- 백업

설정 → **프리미엄 구독**에서 구독할 수 있습니다. Lightning Network를 통해 결제됩니다.
